# 📊 Global Superstore Executive Dashboard

## Overview

This project presents an interactive Executive Sales Dashboard built using Microsoft Power BI. The dashboard helps analyze sales performance, customer behavior, product categories, and regional distribution using the Global Superstore dataset.

---

## Dashboard Preview

![Dashboard](Global%20Superstore.png)

---

## Business Questions

This dashboard answers the following questions:

- What is the total sales performance?
- How many customers and orders are recorded?
- Which product category performs the best?
- Which regions contribute the highest sales?
- How does sales change over time?
- Which Sub-Category generates the largest Sales?
- Which regions have the highest and lowest sales? are there any sales disparities between regions that require special attention?
- Top 10 products with the highest sales?

---

## Dashboard Features

- Executive KPI Cards
- Sales Trend Analysis
- Sales by State
- Sales by Category
- Sales by Segment
- Interactive Slicers
